The second task is about knowledge-aware item-to-item recommendations. We still use DKN for demonstration. The learning framework is illustrated as follows:

To build an item2item recommendation model based on the Recommender repo, you only need to modify two files:

  1. Data Loader : dkn_item2item_iterator.py

  2. Model : dkn_item2item.py

input_creator = DKNItem2itemTextIterator
hparams.neg_num=9

A special parameter is neg_num. It indicates how many negative instances exist in a group for softmax computation. Training and validation instances are organized as follows:

model = DKNItem2Item(hparams, input_creator)

t01 = time.time()
